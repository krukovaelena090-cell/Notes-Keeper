# Notes Keeper 📝

Мини-приложение для ВКонтакте, которое позволяет создавать, редактировать и удалять личные заметки с автоматической синхронизацией между устройствами.

🔗 **Демо:** [https://notes-keeper-bice.vercel.app](https://notes-keeper-bice.vercel.app)  
🆔 **ID приложения VK:** `54475984`

---

## ✨ Функциональность

- ✅ Создание заметок с заголовком и текстом
- ✅ Редактирование и удаление записей
- ✅ Автоматическая синхронизация через Firebase
- ✅ Адаптивный интерфейс на VKUI
- ✅ Персонализация (имя и аватар пользователя ВК)
- ✅ Мгновенное обновление списка заметок

---

## 🛠 Технологический стек

| Технология | Назначение |
|------------|------------|
| **React** + **TypeScript** | Интерфейс и логика |
| **VKUI** | Библиотека компонентов в стиле ВК |
| **VK Bridge** | Взаимодействие с платформой ВКонтакте |
| **Firebase (Firestore)** | Облачное хранение и синхронизация |
| **Vite** | Сборка проекта |
| **Vercel** | Хостинг и автоматический деплой |
| **GitHub** | Хранение исходного кода |
